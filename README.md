# ⚔️ Last Dep Ranked — Игровая Рейтинговая Система

Веб-приложение для отслеживания игровой статистики, ведения истории матчей и автоматического расчета ELO-рейтинга игроков в реальном времени. Проект создан как полноценное интерактивное Fullstack-приложение с динамическим интерфейсом.

## 🚀 Основной функционал
*   **Динамический ELO-рейтинг**: Автоматический расчет позиций игроков на основе их результатов.
*   **Кастомная система рангов**: Динамическое распределение уникальных рангов (от Rank C до анимированного Rank Дракон) в зависимости от процента побед и ELO.
*   **Личный кабинет**: Авторизация пользователей через Google OAuth, возможность редактирования БИО и кастомизации профиля (смена аватара).
*   **Панель модератора (Админка)**: Защищенный интерфейс для администраторов (роли Founder/Archivist) с возможностью быстрой фиксации результатов матчей (счета 3:0, 3:1, 3:2 с коэффициентами сложности) и управления пользователями.
*   **История матчей**: Интерактивное модальное окно профиля, отображающее статистику побед/поражений и последние 3 игры против конкретных оппонентов.

## 🛠️ Стек технологий
*   **Frontend**: HTML5 (Семантическая верстка), CSS3 (Flexbox, Grid, CSS Keyframes для анимации свечения).
*   **Backend & DB**: Supabase (PostgreSQL) — хранение профилей, прав доступа и истории игр.
*   **Интеграции**: Google OAuth для безопасного входа игроков.
*   **Архитектура**: Модульный JavaScript (ES6+), асинхронные запросы (Async/Await) к API базы данных.

## 📱 Интерфейс и дизайн
Проект выполнен в агрессивном игровом темном стиле (Dark Gaming UI) с акцентом на неоновое свечение элементов (Neon Glow) при наведении. Верстка полностью адаптирована под мобильные устройства.
